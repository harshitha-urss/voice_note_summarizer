Problem Statement

Voice notes are widely used for recording meetings, lectures, and personal ideas. However, listening to long recordings and extracting important information is time-consuming and inefficient.

This project provides an AI-based solution that automatically converts voice notes into text and generates concise summaries for quick understanding.

System Workflow

  • User uploads a voice note (audio file)
  • Speech-to-Text model converts audio to text
  • Text is cleaned and preprocessed
  • Summarization model extracts key points
  • User receives summarized text output
